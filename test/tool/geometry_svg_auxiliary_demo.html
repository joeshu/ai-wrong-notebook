几何配图 SVG + 辅助线动态演示

点击"显示辅助线"切换辅助线的显示/隐藏。
点击"动画展示"可以看到辅助线逐步绘制的动画效果，模拟解题思路的展开过程。

样例 1：三角形内角和（外角辅助线）

如图，在△ABC中，∠A = 50°，∠B = 65°，求∠C的度数。
辅助线思路：延长 BC 到 D，利用外角定理。

样例 2：半圆面积（半径辅助线）

如图，半圆的直径 AB = 10 cm，求阴影部分（半圆）的面积。
辅助线思路：连圆心 O，标注半径 r = 5。

样例 3：等腰三角形底角（作高辅助线）

如图，等腰三角形 ABC 中，AB = AC，∠A = 40°，求∠B。
辅助线思路：作底边 BC 的中线 AD（也是高），将等腰三角形分为两个全等直角三角形。

样例 4：平行四边形面积（作高辅助线）

如图，平行四边形 ABCD 中，AB = 8，∠B = 60°，BC = 5，求面积。
辅助线思路：从 A 作 BC 的高 AE，利用三角函数求高。

样例 5：圆内接角（直径辅助线）

如图，AB 是圆 O 的直径，C 是圆上一点，求∠ACB。
辅助线思路：连接 OC，利用"直径所对的圆周角是直角"定理。

技术说明

  • 辅助线动态展示：CSS animation + stroke-dashoffset 实现"画线"效果，text 用 fadeIn 延迟出现
  • 交互方式：checkbox 切换显示/隐藏，按钮触发动画（模拟解题思路逐步展开）
  • Flutter 对应：AnimationController + CustomPainter 可实现相同效果
  • 配色：主图 #333，标注 #1a73e8，已知条件 #e63946，辅助线 #f57c00
