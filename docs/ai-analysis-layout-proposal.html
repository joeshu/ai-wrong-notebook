1. 已知 x² + 1 = 5，求 x 的值。OCR识别文本仅用于核对。

正确答案

由 x² + 1 = 5 得 x² = 4，所以 x = ±2。

错因分析

学生常见错误有三类：一是把 x² = 4 只解成 x = 2，遗漏 x = -2；二是对“开平方”理解不清，没有意识到平方为 4 的数有两个；三是缺少检验步骤，导致漏解后没有发现。

学习建议

遇到含有平方的方程时，先化成标准形式，再判断是否需要开平方。特别要记住：若 x² = a 且 a > 0，那么 x = ±√a。

知识点

只保留一处
解形如 x² = a 的方程时，若 a > 0，则有两个实数解：x = ±√a。
平方与开平方是互逆运算，但开平方时必须注意正负两种情况，不能漏解。
解方程后要养成代入原方程检验的习惯，确认所得结果是否都成立。

解题步骤

1

先将方程 x² + 1 = 5 两边同时减去 1，得到 x² = 4。

2

再对等式两边开平方，注意平方根有正负两种情况，所以 x = ±2。

3

最后检验：当 x = 2 时，2² + 1 = 5；当 x = -2 时，(-2)² + 1 = 5，两个值都满足原方程。

5. 如图所示的实例中，与大气压有关的是（ ）A. 医生进行肌肉注射 B. 下水管 U 形设计 C. 用吸盘搬运玻璃 D. 书包带做得宽大

正确答案

C. 用吸盘搬运玻璃。吸盘压在玻璃上时，内部空气被挤出，外部大气压将吸盘压在玻璃表面，从而能够搬运玻璃。

错因分析

学生可能把“压强”和“大气压”混淆。A 主要利用液体压强和推力，B 主要利用连通器原理，D 是通过增大受力面积来减小压强，只有 C 直接利用大气压。

学习建议

判断生活实例是否与大气压有关时，重点看是否存在“内部压强变小、外部大气压把物体压住或托住”的现象。

知识点

只保留一处
大气压会对浸在空气中的物体产生压强，生活中常见现象有吸盘、吸管吸饮料、注射器吸药液等。
吸盘工作原理：挤出内部空气后，内部压强减小，外部大气压将吸盘压在物体表面。
压强公式 p = F / S：受力面积越大，压强越小；书包带做宽是为了减小压强，不是大气压现象。

解题步骤

1

分析 A：医生进行肌肉注射，主要利用液体压强和推力使药液进入人体，不是大气压现象。

2

分析 B：下水管 U 形设计，主要利用连通器原理并起到存水防臭作用，不是大气压的典型应用。

3

分析 C：吸盘内部空气减少，外部大气压把吸盘压在玻璃表面，这是典型的大气压应用。

4

分析 D：书包带做宽是为了增大受力面积、减小压强，属于压强知识。

5

综合判断，正确选项是 C。
