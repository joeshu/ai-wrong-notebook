Q
原题
已知抛物线 y = ax² + bx + c 经过 A(1,0)、B(3,0)，且顶点纵坐标为 -4，求解析式。
✓
正确解答
1由零点 1、3，设 y = a(x - 1)(x - 3)。
2对称轴 x = 2，顶点代入得 -a = -4，所以 a = 4。
3解析式为 y = 4x² - 16x + 12。
!
错因分析

你把顶点的横坐标和纵坐标混在一起使用，导致代入位置错误。先确定函数形式，再用顶点条件求参数会更稳定。

★
学习建议

遇到已知零点的二次函数，优先使用交点式；遇到顶点条件时，再转入顶点坐标或代入对称轴。
